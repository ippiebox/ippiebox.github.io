DayMark 1.0.7

DayMark the date, and know the date distance from today.
It is very easy to use, check it out!

Main Screen

At the top, there are four blue buttons

Edit - Rearrange the order, or delete any event
Search - Incremental search
Favorite - Show only favorite events
Add - Add new event

Tap the hidden button DayMark to show the Preference settings

Edit Event Order / Delete Event

After tapping the Edit blue button

You may tap the red button on the left to delete event

Or press and hold the 3-lines button on the right to rearrange order

Tap Done blue button when complete

Search

After tapping the Search blue button

Type in what you are looking for 

The matching events would be shown with highlight 

Tap Cancel blue button when done

Show Favorite Events

Tap the Favorite blue button to toggle the favorite mode



Add New Event

After tapping the Add blue button

Type in the New Event name

Tap the Star to toggle favorite event 

Set the Date using the Date Picker

The Slider on top of the Date Picker allows quick date adjustment. Slide to the left to shift the date to the past, or slide to the right for  future date

Tap the blue Today button to jump back to today

Tap Save when complete, or Cancel to finish without saving 

Preferences

Tap the DayMark title button to change the settings

1) Set day count in number (e.g. In 123 days) or calendar format (e.g. 1 years 6 months 4 days ago)

2) Turn on/off the notification for today's events 

3) Enable/Disable Today alert pop up after opening the app

4) Add new event to the top or bottom

Tap Save when done, or Cancel to finish without saving 

Tap ippiebox logo at the bottom to visit our website
